📋 使用步驟

  1. 啟動伺服器：npm start
  2. 啟動 ngrok：ngrok http 3000
  3. 複製 ngrok 網址（例如：https://abc123.ngrok.io）
  4. 生成QR碼：在上方輸入完整網址
  5. 手機測試：用手機掃描QR碼

🔧 故障排除

  • QR碼無法掃描：檢查網址是否正確
  • 頁面無法載入：確認 ngrok 是否正常運行
  • 手機無法連接：檢查網路設定
